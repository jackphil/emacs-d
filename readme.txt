需要手工修改的地方
====

虽然作了很大努力实现自动话，但有些地方还是需要你手工修改：

init.el: section 8关于邮件部分

site-lisp/misc/mswin.el: 设置buffer默认目录中的目录名

".authinfo": 邮件认证信息，安全原因，不建议使用

gnus/gnus.el: 帐户设置（pop地址，用户名，密码等）

mswin/emacsclientw.reg: 几个目录设置

Windows
====

推荐使用emacsclientw，更多内容或使用gnuclient(w) 参见http://jackphil.blogspot.com/2008/11/emacsclientwgnuclientw.html
